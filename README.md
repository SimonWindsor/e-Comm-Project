# Daintree Store Backend

This is the backend API for the Daintree full-stack e-commerce application. It is currently under active development and provides the data layer for the frontend.

## Tech Stack
- Node.js
- Express.js
- PostgreSQL
- REST API

## Current Functionality
- User authentication (register / login)
- Product data retrieval
- Cart persistence
- Reviews and purchases (partial implementation)

## Running Locally

To run this project locally, you will need to configure environment variables:

DATABASE_URL=your_database_connection_string
SESSION_SECRET=your_session_secret
FRONTEND_URL=http://localhost:3000

Then run:

```bash
npm install
npm run dev
```
